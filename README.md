# Calculadora em Front-End
Projeto calculadora

## Descrição
Calculadora simples usando HTML, CSS e JavaScript.  
Permite somar, subtrair, multiplicar e dividir números.

## Funcionalidades
- Operações: +, -, *, /
- Botão C para limpar o display
- Botão = para calcular
- Atualiza o display em tempo real

## Como usar
1. Abra a pasta `calculadora`.
2. Abra o arquivo `index.html` no navegador.
3. Clique nos números e operadores para realizar cálculos.

## Tecnologias
- HTML
- CSS
- JavaScript
